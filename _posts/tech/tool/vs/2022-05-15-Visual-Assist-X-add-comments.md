单击Suggestions -> Edit VA Snippets，打开VA Snippet Editor页面。

在VA Snippet Editor 页面下便可添加Doxygen注释模板。
```
 
/**
* @brief 
* 
* 
* 
* @param 
* @param 
* @return 
* @date $YEAR$-$MONTH$-$DAY$
* @see 
*/
```
