Dependency Manager for PHP

https://getcomposer.org/

## 安装
curl -sS https://getcomposer.org/installer | php -- --install-dir=/usr/local/bin --filename=composer
composer
注："php -- --install-dir"2个--之间有空格

## 常见语言的包管理工具
PHP => Composer
Ruby => RubyGems
Nodejs => npm
Java => Marven( mvn )
Python => pip
