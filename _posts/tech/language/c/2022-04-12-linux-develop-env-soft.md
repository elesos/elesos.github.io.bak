## CentOS

```
yum update 
yum install -y autoconf automake bzip2 bzip2-devel cmake freetype-devel gcc gcc-c++ git libtool make mercurial pkgconfig zlib-devel nasm vim wget lrzsz tar unzip subversion 
```

## Ubuntu

```
apt update
sudo apt  -y install autoconf automake build-essential cmake git-core libtool pkg-config wget vim openssl libssl-dev  libpcre3 libpcre3-dev
```
