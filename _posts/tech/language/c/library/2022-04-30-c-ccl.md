配置文件为键值对形式key=value

## 安装
```
wget https://github.com/opensourceworker/assets/raw/master/2019/06/ccl-0.1.1.tar.gz
tar -zxf ccl-0.1.1.tar.gz
./configure --prefix=/usr/local
make  && make install
```
