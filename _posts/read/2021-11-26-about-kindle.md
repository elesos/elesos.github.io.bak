# Kindle支持哪些格式

官方产品介绍页面有相关技术参数：

```
Kindle Format 8 (AZW3)，
Kindle (AZW)，
TXT，PDF，
MOBI，
PRC原格式，HTML，DOC，DOCX，JPEG，GIF，PNG，BMP转换格式。
```

不过不支持 epub 格式，需要自己转换，可在下面参考链接中找到相关转换工具。

# 在Kindle设备上使用微信读书

1. Kindle 设备不支持安装第三方应用，只能使用微信读书网页版。
2. 点击Kindle右上角，然后打开「体验版网页浏览器」。输入

```
r.qq.com
```

即可
