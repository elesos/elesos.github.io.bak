◆ 自序

>> 1997年，我开始业余写作不久，就写了一篇散文《解剖曾国藩》

>> 曾国藩是一个钻之弥坚、仰之弥高的人物


◆ 第一章 曾国藩的七次科举之痛

>> 中国古代的科举制分三步：秀才、举人、进士。

>> 行年八十尚称童，可云寿考；到老五经犹未熟，真是书生。”￼意思是说：都八十岁了，还是个“童生”，可见你将来能活个几百岁；都到老了五本经书还没背熟，你真可谓“书生”。

>> 往还湘潭市肆，与裘马少年相逐

>> 凭什么别人家能供出举人进士，我们曾家就供不出来？儿子供不出来，我还有孙子，孙子供不出来，我还有重孙子，我就不信曾家没有翻身的一天！

>> 秀才考试分为三关：县试、府试和院试（也就是省一级考试）

>> 官呵吏骂

>> 看过考题后，曾国藩并没有像以前那样急于下笔，而是先默坐思考了半个时辰，在胸中构思好了整体脉络，然后才一股一股地展开，最后在文笔上细细打磨

>> 中了秀才之后第二年，恰是乡试之年，

>> 晨夕讲授，指画耳提

>> 梁启超更是天资超迈，十一岁中秀才，十六岁中举人


◆ 第二章 为什么要“学做圣人”

>> 顾盼自雄

>> 青年时代，曾国藩的全部精力都用在八股文上，除了四书五经之外没读过什么书

>> 窍被茅塞久矣

>> 曾国藩反思刚才自己的言谈举止，感觉脸上有点儿发烧。

>> 神钦鬼服

>> 曾国藩学做圣人的方法很简单，就是“写日记”。

>> 记日记最重要的目的是反省自己。

>> 闲游荒业”“闲谈荒功

>> 曾国藩为什么把日记给朋友看呢？因为外力远远大于内力

>> 失乳彷徨”￼。
把戒烟喻为婴儿断乳，可谓相当准确

>> 要改掉其他缺点，则远不如戒烟那么容易。

>> 自新之意安在？”

>> 首尾不懈，不可见异思迁

>> 立志”或者说确立一个终生的奋斗目标，对一个人的精神成长是至关重要的


◆ 第三章 惊人的进阶之道

>> 给自己立下十二条做人的规矩和课程的标准：

>> （2）静坐。

>> （3）早起。

>> 读书不二。一书未完，不看他书。

>> 凡事不留待明日，愈积愈难清。

>> 经常交往的朋友有一百多人。最多的一天，接待或者拜访朋友几十人

>> 一个人的声望是晋升的重要基础


◆ 第四章 从前的官场愣头青

>> 在紧张的工作之余，曾国藩仍“手不释卷”，只不过更注重实用了

>> 自憎形影

>> 去国情如失乳儿

>> 不能以自效，退不能以自存

>> 小不见大，小事精明，大事糊涂

>> 察于小事，而且有的谨慎到不该谨慎的程度，而对于国家的大计却没有时间和精力去深究。

>> 待其弊既成而后挽之，则难为力矣。”

>> 官官相护”的潜规则

>> 成熟，要心胸开阔，要辩证地、全面地看问题。天塌大家死，我何必独着急？


◆ 第五章 “曾剃头”的长沙之辱

>> 祝祷神明，祈佑兄体。

>> 带着自己募来的湘军，前往僻静的衡阳

>> 不争一日之短长。等到我在衡阳练成一支劲旅，打几个胜仗给你们看看，那时自会分出高下。这才是挽回面子的最好办法！


◆ 第六章 湘军为什么牛气

>> 秦始皇焚书坑儒之后，没有比洪秀全更厉害的文化毁灭者。

>> 清代国家正规军分八旗和绿营两部分。八旗是满族武装，绿营是汉人军队

>> 惜我兵一百名如见鹯之雀，一百勇如裹足之羊

>> 三是还没有经验。如何招兵，如何筹饷，如何采购武器，如何编营，如何训练，如何制定营规，如何选择军官，如何设定军饷标准，如何排兵，如何布阵……太多环节要从头学起。

>> 湘军和正规军最大的不同是“不食于官”。不必国家拨给经费，而是像团练那样钱粮自行筹措。这是湘军唯一像团练的地方。

>> 没有经验，曾国藩就自己在黑暗中摸索。在具体实践过程中，曾国藩不断失败，不断犯错。我们不谈陆军，先来看看曾国藩是如何创建水师的。

>> 剑戟不利，不可以断割；毛羽不丰，不可以高飞

>> 舟师则船炮并富，陆路则将卒并愤，

>> 草率从事，驱不教之士，执窳脆之器，行三千里

>> ，致使江、吴先后兵败自杀，作为省会的庐州、武昌也迭遭沦陷。此时的曾国藩承受的心理压力可想而知。但从战略上看，曾国藩这两次抗疏拒旨，则为造船、购炮与练兵赢得了时间，使湘军免遭轻进覆灭之灾，为日后彻底战胜太平天国积蓄了基本力量。

>> 群疑众谤

>> 湘潭之战，湘军水陆不足万人，与三万之众的太平军做殊死战，前后六天，十战十捷，以少胜多，毙敌近万，太平军逃散者也上万，太平军精锐林绍璋部几乎全军覆灭。

>> 可以说，没有长沙之辱，就没有曾国藩后半生的功名事业

>> ，对于有志者来说，挫辱是最大的动力，打击是最好的帮助。咬紧牙关，把挫辱活生生吞下，就成了滋养自己意志和决心的营养

>> 我在这句话里所说的“反思”，用的字面意思，也就是反向思考

>> 国藩在创建湘军的时候，并没有首先去想“湘军怎么才有战斗力”，而是先来分析“绿营兵为什么没有战斗力”。这是一个很重要的切入点。其实很多时候做事不需要你有什么超天才的想法。你只要好好总结一下别人都犯了哪些错误，然后你反着来，你可能就成功了。

>> 每一层上级和下级之间都知根知底，且有选用提拔之恩

>> 那么曾国藩为什么偏要选没有经验、手无缚鸡之力的读书人呢？这是因为曾国藩一贯认为，精神的力量远大于身体的力量

>> 绝对不收当过兵的人，不收退伍军人。

>> 近山者仁，近水者智

>> 山僻之民多犷悍，水乡之民多浮滑，城市多游惰之习，乡村多朴拙之夫

>> 真正厉害的将军，不仅要搞物质刺激，还要抓住这支军队的心灵。

>> 国藩把军队的训练，分成了“训”和“练”这两个部分。所谓“训”，就是“训话”，也就是政治思想教育

>> 其过人之处就是不怕费心费力，对事物进行不留死角的深入分析。在对事物进行精到分析的基础上，再找出要害，把握关键。每次处理完了之后，还要总结经验教训，为下一次做参考。


◆ 第七章 得而复失的湖北巡抚

>> 功虽大而不喜，过虽小而必究

>> 一次失败都要吃透，才能不白白付出这个代价

>> 凡是临阵脱逃的将领，不管你当时有什么理由，都立刻打发回家，永远不再录用。

>> 曾国藩还为湘军立下这样一条规矩：一旦某军统帅战死，除非本部有受拥戴的继任将领，否则必全军遣散回乡，另行招募成军。这样一来，部下打仗的时候都会全力保护自己的长官。因为只有保住长官，你才有个人发展的机会

>> 曾国藩高明就高明在他用制度解决了别人用道德去解决的问题。保卫长官本来是一种道德要求，但曾国藩通过制度使它变成了符合下属自身利益的一种行为。

>> 任何政治人物都不可能没有敌人，关键是树这个敌要值得

>> 大清王朝各省的实权掌握在省长们也就是总督和巡抚们的手

>> 然而，不管皇帝对他如何，他已经开始的军事生涯，还是得继续下去。因为他与太平天国的斗争，不仅是为了皇帝，更是为了天下，为了文化道统。

>> 所谓战略，就是基于宏观性和长远性，把各种纷繁复杂的要素按轻重缓急、先后次序进行整理排列，然后拿出最有效的行动方案。

>> 自古平江南之策，必踞上游之势，建瓴而下，乃能成功


◆ 第八章 江西困境与“大悔大悟”

>> 湖口之败后，湘军元气大伤

>> 借“潘驴邓小闲”之典。

>> 一言不妄发，一步不妄行

>> 芒鞋

>> 在军中成天与罗泽南讲道学。虽然相处时间不长，却与他建立起极深的情谊。

>> 原湖北巡抚陶恩培因为城破自杀，朝廷任命胡林翼当了湖北巡抚，成了独当一面的封疆大吏。

>> 咸丰如此破格提拔，相信胡林翼一定会感恩戴德。￼
这就是咸丰的用人术

>> 不放罗泽南，武汉确实一时难以拿下来；放走罗泽南，曾国藩自身安危难保。
但是一番犹豫之后，曾国藩还是派出了罗泽南。因为从当时天下大局看，武汉确实更为重要，自己的安危则为次之

>> 国藩在关键时刻之顾全大局的胸怀实为普通人之所不及。

>> 曾氏兄弟五人，曾国藩居长，下面有四个弟弟，大弟曾国潢，二弟曾国华，三弟曾国荃，四弟曾国葆

>> 就在曾国华从军后数月，另一个弟弟曾国荃也弃文从武，率兵前往江西救援

>> 被皇帝拿掉兵权

>> 窘若拘囚；群疑众侮，积泪涨江

>> 咸丰皇帝为什么如此果断地罢了曾国藩的军权呢？
因为他另有了依靠。

>> 劳师糜饷

>> 蛰伏两年，完成“脱胎换骨”

>> 湘军攻陷九江，杨载福、李续宾皆赏穿黄马褂

>> 曾国藩反思到，自己在官场上一再碰壁，碰得鼻青脸肿，不光是皇帝小心眼，大臣多私心，自己的个性、脾气、气质、风格上的诸多缺陷，也是重要原因。回想自己以前为人处世，总是怀着强烈的道德优越感，自以为居心正大，人浊我清，因此高己卑人，锋芒毕露，说话太冲，办事太直，当然容易引起他人的反感

>> 行有不得，反求诸己”这句圣人之言，他虽然耳熟能详，实际上没有真正做到过。

>> 担夫争道悍妇诟室

>> 有些太特立独行了。

>> 太自傲、太急切、一味蛮干、一味刚强。

>> 行事过于方刚者，表面上似乎是强者，实际上却是弱者。

>> 天下之至柔，驰骋天下之至坚

>> 大柔非柔，至刚无刚”。

>> 有始无终，怠慢简脱；

>> 慎终如始，

>> 官场生存，必须习惯官场上虚与委蛇的那一套

>> 原来对那些无用的官样文章，他不理不睬，现在则每信必复

>> 他认识到“不妄保举，不乱用钱”，则“人心不附”。只有诱之以“名”，笼之以“利”，才能网罗天下英才。


◆ 第九章 安庆这块难啃的骨头

>> 胡林翼的性格特点是明敏、坚毅、精于权术。此时他已经以极为高明的手腕，笼络和操纵住了湖广总督官文。

>> 林翼与官文刻意交好，凡有功劳，皆推让于官文。他还千方百计讨好官文的小妾，让自己的母亲收其为义女

>> 庐州就是今天的合肥

>> 太平军抓住机会发起围攻，李续宾六千精锐被全歼，李续宾及曾国藩的亲弟弟曾国华皆死于军中。

>> 从来攻城时，未见今日武”。￼

>> 比分肥，贪污中饱。

>> 骄兵惰，终日酣嬉，不以贼匪为意。或

>> 淫心荡志，极乐忘疲。

>> 以多人张虚声

>> 湘军行军作息表，是每天要花四小时行军，走三十里。然后就不走了，干什么呢？再花四小时挖沟修墙。

>> 气常静，客气常动。客气先盛而后衰，主气先微而后壮。故用兵者喜为主，不喜作为客。

>> 。曾国藩的四大弟子之一薛福成

>> 江苏、浙江向来是清政府的主要赋税来源地，

>> 战略上有一条基本的原则，就是致人而不致于人，也就是要迫使对方按照自己的战略安排行动，迫使对方跟着我们的步子走，而不是按照对方的战略安排行动。说白了，就是将战略的主动权掌握在自己的手中。

>> 此事无益于北，有损于南。”

>> 多隆阿自然非常生气，安庆之战结束后就和曾氏兄弟生了嫌隙，不久之后他离曾国藩而去。

>> 国之诚，进德之猛，好贤之笃，驭将之厚，察吏之严，理财之精，何美不备？何日不新？


◆ 第十章 与何桂清的恩怨纠葛

>> 他在用当年王有龄、何桂清逼死罗遵殿的办法，来收拾王有龄

>> 曾国藩的成功，证明了传统文化的强大生命力，可以部分抵消对儒学“空疏无用”的指摘。曾国藩一生功业都是在传统文化的支撑下完成的。以天下为己任的强烈使命感，修身齐家治国平天下的宏大志向，民胞物与的博大胸怀是支撑曾国藩在艰难困苦中奋力挣挫的精神动力。实事求是、经世致用、反身而诚的认知传统又使他能够从前人，从他人，从自身，学到智慧和经验，应对复杂的世事，饶有余裕。至诚待天、忠恕对人的道德准则，使得他能为人磊落，不为低级趣味所纠缠，纳人细垢，成己大德，用自己的人格力量去降服人，吸纳人，使英雄为我所用，终成大业。

>> 他说：“学于古，则多看书籍；学于今，则多觅榜样。”

>> 曾国藩事君至忠，事亲至孝。

>> 对于儿女，他既慈爱又严格，能够尊重孩子人格，教育孩子总以鼓励为主，没有传统家长那种居高临下的不平等作风。
